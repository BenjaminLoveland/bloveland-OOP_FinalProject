ai.py

Module to handle interactions with ChatGPT for Connect Four AI decision-making.

Class: ConnectFourAI

Handles AI decision-making for the Connect Four game.

  • Methods:
    • __init__(api_key: str) -> None - Initializes the AI with an OpenAI API key.
    • get_best_move(board: List[List[int]]) -> int - Determines the best move for the AI based on the game board.

app.py

Module to run the Connect Four application using Flask.

Flask Routes

  • index() -> str - Renders the index page for the Connect Four game.
  • start_game() -> Response - Starts a new Connect Four game.
  • get_move() -> Response - Calculates and applies the AI's best move based on the current board.
  • check_winner() -> Response - Checks if there is a winner in the current game.

Function: main()

Runs the Flask application in debug mode.

game.py

Module to handle gameplay logic for Connect Four.

Class: Game

Represents the gameplay logic for Connect Four.

  • Properties:
    • rows() -> int - Gets the number of rows in the board.
    • columns() -> int - Gets the number of columns in the board.
    • board() -> List[List[int]] - Gets the game board.
    • board(new_board: List[List[int]]) -> None - Sets the game board.
    • current_player() -> int - Gets the current player.
    • current_player(player: int) -> None - Sets the current player.
  • Methods:
    • __init__() -> None - Initializes the game board and sets the current player.
    • reset_board() -> None - Resets the game board to its initial state.
    • check_winner() -> int - Checks if there is a winner. Returns 1 or 2 if a player wins, 0 otherwise.
    • make_move(column: int) -> Tuple[int, int] - Makes a move for the current player in the specified column.
    • is_column_full(column: int) -> bool - Checks if a specified column is full.
    • to_string() -> str - Returns a string representation of the game board.
